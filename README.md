# 🚀 CodeQuest — Online Coding Practice Platform

**CodeQuest** is a full-stack web application inspired by platforms like LeetCode and HackerRank. It enables users to practice data structures and algorithms through a variety of coding challenges across different difficulty levels. The platform supports user authentication, problem-solving in real-time, and tracking of submissions.

## 🔧 Tech Stack

- **Frontend:** React.js (with Hooks, Context API)
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **ORM:** Prisma / Sequelize (optional)
- **Authentication:** JWT (JSON Web Tokens)
- **Deployment:** (Add once deployed - e.g., Vercel, Render, Railway)

---

## 📌 Features

- 🧑‍💻 User Registration & Login (JWT-based authentication)
- 📚 Browse problems by topic, difficulty, and tags
- ✍️ Code editor with syntax highlighting
- ✅ Submit code and view results (pass/fail test cases)
- 📊 Track submissions and performance history
- 🔍 Search functionality for problems
- 🔐 Secure backend with route protection
